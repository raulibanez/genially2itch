# Genially2Itch

Herramienta web para convertir presentaciones de Genially en archivos ZIP listos para subir a [itch.io](https://itch.io) en modo HTML.

## Uso

1. Accede a la herramienta: [https://raulibanez.github.io/genially2itch](https://raulibanez.github.io/genially2itch)
2. Introduce la URL de tu Genially (ejemplo: `https://view.genially.com/XXXXXXXXXXXX`)
3. Pulsa "Generar ZIP"
4. Sube el ZIP a itch.io como proyecto HTML

## Configuración en itch.io

Al subir el ZIP a itch.io, asegúrate de:
- Seleccionar **"This file will be played in the browser"**
- Marcar el tipo de proyecto como **HTML**

La herramienta incluye una imagen de referencia con la configuración recomendada.

## Demo

Puedes ver un ejemplo de un Genially subido a itch.io aquí:
